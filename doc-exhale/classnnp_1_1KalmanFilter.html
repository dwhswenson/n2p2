class KalmanFilter : public nnp::Updater¶

Implementation of the Kalman filter method.

Public Types

enum KalmanType¶

Enumerate different Kalman filter types.

Values:

KT_STANDARD¶

Regular Kalman filter.

KT_FADINGMEMORY¶

Kalman filtering with fading memory modification.

enum KalmanParallel¶

Enumerate different parallelization modes.

Values:

KP_SERIAL¶

Serial Kalman filter, no parallelization.

KP_NBCOMM¶

Non-blocking communication, parallel matrix product P * H.

KP_PRECALCX¶

Precalculate intermediate X = P . H for each stream.

void setParametersStandard(double const epsilon, double const q0, double const qtau, double const qmin, double const eta0, double const etatau, double const etamax)¶

Set parameters for standard Kalman filter.

Parameters
  • epsilon: Error covariance initialization parameter \(\epsilon\).
  • q0: Process noise initial value \(q_0\).
  • qtau: Process noise exponential decay parameter \(q_{\tau}\).
  • qmin: Process noise minimum value \(q_{\text{min}}\).
  • eta0: Initial learning rate \(\eta_0\).
  • etatau: Learning rate exponential increase parameter \(\eta_{\tau}\).
  • etamax: Learning rate maximum value \(\eta_{\text{max}}\).

void setParametersFadingMemory(double const epsilon, double const q0, double const qtau, double const qmin, double const lambda, double const nu)¶

Set parameters for fading memory Kalman filter.

Parameters
  • epsilon: Error covariance initialization parameter \(\epsilon\).
  • q0: Process noise initial value \(q_0\).
  • qtau: Process noise exponential decay parameter \(q_{\tau}\).
  • qmin: Process noise minimum value \(q_{\text{min}}\).
  • lambda: Forgetting factor \(\lambda\).
  • nu: Fading memory parameter \(\nu\).
