Detailed Description

Radial symmetry function (type 2)

Parameter string:

<element-central> 2 <element-neighbor> <eta> <rshift> <rcutoff>

where

  • <element-central> .... element symbol of central atom
  • <element-neighbor> ... element symbol of neighbor atom
  • <eta> ................
  • <rshift> .............
  • <rcutoff> ............
