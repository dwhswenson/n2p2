InterfaceLammps()¶
void initialize(char *const &directory, bool showew, bool resetew, int showewsum, int maxew, double cflength, double cfenergy, double lammpsCutoff, int lammpsNtypes, int myRank)¶

Initialize the LAMMPS interface.

Parameters
  • directory: Directory containing NNP data files (weights, scaling, settings).
  • showew: If detailed extrapolation warnings for all atoms are shown.
  • resetew: If extrapolation warnings counter is reset every timestep.
  • showewsum: Show number of warnings every this many timesteps.
  • maxew: Abort simulation if more than this many warnings are encountered.
  • cflength: Length unit conversion factor.
  • cfenergy: Energy unit conversion factor.
  • lammpsCutoff: Cutoff radius from LAMMPS (via pair_coeff).
  • lammpsNtypes: Number of atom types in LAMMPS.
  • myRank: MPI process rank (passed on to structure index).
